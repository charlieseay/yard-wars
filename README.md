# Yard Wars 🎯

**Friendly games. Serious stakes.**

Track side games, bets, and custom card modifiers for outdoor activities like disc golf, cornhole, horseshoes, and more.

---

## Quick Start

```bash
./scripts/test-automation.sh
```

Runs automated testing suite with health checks. See [AUTOMATED-TESTING.md](AUTOMATED-TESTING.md) for details.

---

## Features

- **Side Games Tracking** - Skins, chips, card-based modifiers
- **Custom Game Builder** - Create your own rules and decks
- **OLED Optimized** - True black (#000000) for battery savings
- **Offline-First** - Zero network calls, 100% local storage
- **QR Sync** - Peer-to-peer scorecard sharing

---

## Tech Stack

- **Platform:** iOS + Android (React Native / Expo SDK 57)
- **Architecture:** Local-first, offline-only, zero-backend
- **Storage:** Atomic JSON writes + SQLite
- **Theme:** OLED optimized (true black + neon colors)
