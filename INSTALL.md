# Install Yard Wars on iPhone

## Quick Install

**Expo dev server is running on your Mac.**

### On your iPhone:

1. Open **Camera** app
2. Point at this QR code (check terminal window or http://localhost:8081)
3. Tap the notification
4. App opens in **Expo Go**

---

## What You'll See

- **New Yard Wars logo** (disc golf basket shield)
- **Splash screen** with "Friendly games. Serious stakes."
- **Health check auto-runs** on first load (watch console)
- **Setup screen** ready to start a round

---

## Health Check

App automatically validates on launch:
- ✅ File System API (Expo SDK 57)
- ✅ SQLite API
- ✅ Storage read/write

Results appear in console immediately.
